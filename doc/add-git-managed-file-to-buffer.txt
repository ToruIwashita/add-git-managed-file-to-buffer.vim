				*add-git-managed-file-to-buffer-introduction*
Introduction

*add-git-managed-file-to-buffer* provides the ability to add files that are
changed, modifiled, untracked in Git to the buffer.

==============================================================================
				*add-git-managed-file-to-buffer-interface*
Interface

------------------------------------------------------------------------------
				*add-git-managed-file-to-buffer-commnads*
Commands

								*:AddCToBuffer*
:AddCToBuffer
	Add all changed files to the buffer.

								*:AddMToBuffer*
:AddMToBuffer
	Add all modifiled files to the buffer.

								*:AddUToBuffer*
:AddUToBuffer
	Add all untracked files to the buffer.

								*:AddCToTab*
:AddCToTab
	After emptying the buffer, add the changed file to the buffer and open
	it in the tab.

								*:AddMToTab*
:AddMToTab
	After emptying the buffer, add the modifiled file to the buffer and
	open it in the tab.

								*:AddUToTab*
:AddUToTab
	After emptying the buffer, add the untracked file to the buffer and
	open it in the tab.

							*:OpenAllBuffersInTab*
:OpenAllBuffersInTab
	Open all buffers in tab.
